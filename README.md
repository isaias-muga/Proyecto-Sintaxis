# Proyecto-Sintaxis
Proyecto Sintaxis
**1. Descargar el proyecto (Solo la primera vez)**
Abrí la terminal donde quieras guardar el TP y ejecutá:
`git clone https://github.com/isaias-muga/Proyecto-Sintaxis.git`

**2. Entrar a la carpeta**
`cd Proyecto-Sintaxis`

**3. Actualizar tu compu con lo último de GitHub (Antes de empezar a trabajar)**
Siempre ejecutá esto para no pisar el código del otro:
`git pull origin main`

**4. Subir tus cambios (Cuando termines)**
`git add .`
`git commit -m "Mensaje explicando qué hiciste"`
`git push`
*(Nota: GitHub te pedirá tu usuario y un Personal Access Token la primera vez que hagas push).*
